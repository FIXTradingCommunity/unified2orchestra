### Codeset ResetSeqNumFlagCodeSet type Boolean (141)

Indicates that both sides of the FIX session should reset sequence numbers.

| Name | Value | Id     | Sort | Synopsis                    |
|------|-------|--------|------|-----------------------------|
| No   | N     | 141001 | 1    | No                          |
| Yes  | Y     | 141002 | 2    | Yes, reset sequence numbers |
